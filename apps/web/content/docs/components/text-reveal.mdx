## Props

| Prop         | Type   | Description                                  | Default |
| ------------ | ------ | -------------------------------------------- | ------- |
| children     |        | The text to be shimmered.                    |         |
| className    | string | The class name to be applied to the shimmer. |         |
| text         | string | The text to do the animation for             | ""      |
